Sweet Eat's Bakery

This is the second project I completed with Lambda School. The first was learning semantic HTML for this site. This features some gentle animations, and an additional "Weddings" page.

User Interface

My first Lambda GitHub project repository for a webpage, this project features a fixed navigation bar as you scroll, and a fun animation on the logo. The focus was learning HTML and CSS. It is non-responsive, and styled with inline-block; we were not permitted to use flex-box for this iteration of the site.

Flex Box

Finally! After the struggle to style the Great Idea's webpage in the previous "User Interface" project, we refactored this project using flex-box. We also learned that refactoring an earlier project to work with different properties can be quite...interesing.

User Interface & Git

This independent, timed project tested our HTML & CSS skills, as well as our appropriate usage of Git. We built the Navigation and Header, and set colors to boxes. We then built an About page. I also worked on some stretch goals for this project.

Responsive Web Design

Here, we revisit the Great Ideas website while learning responsiveness. This is the first project containing Media Queries, and learned about adding a meta tag for responsiveness. I added a responsive Services page, as well.
